[[support-for-aws-govcloud-us-accounts]]
=== Prise en charge des comptes AWS GovCloud (US)

Vous pouvez désormais créer des comptes AWS GovCloud (US) dans Workload Factory. Ce type de compte utilise uniquement les points de terminaison et les régions AWS US GovCloud pour la configuration et les opérations. Les informations d'identification et les secrets restent dans GovCloud. Les régions GovCloud (US) sont prises en charge.

link:https://docs.netapp.com/us-en/workload-setup-admin/sign-up-saas.html["Créez un compte AWS GovCloud (US) dans Workload Factory"]
